06树

树

二叉树

1.特点

  • 每个节点的度最大为2
  • 左子树和右子树是有顺序的
  • 即使某节点只有一棵子树，也要区分左右子树
  • 只有跟节点的树是二叉树

2.性质

  • 非空二叉树的第i层，最多有==2 ^i-1^ ==个节点（i>=1)
  • 在高度为h的二叉树上最多有==2 ^h^====-1==个节点（h>=1）
  • 对于任何一棵非空二叉树，如果叶子节点个数为n~0~，度为2的节点个数为n2，则有：==n0 = n2+1==
  • 假设度为1的节点个数为n1,那么二叉树的节点总数==n = n0+n1+n2==
  • 二叉树的边数==T=n1+2*n2=n-1=n0+n1+n2-1==

真二叉树

所有节点的度要么是0要么是2。且所有的叶子节点都在最后一层

满二叉树

最后一层节点的度都为 0，其他节点的度都为 2

  • 假设满二叉树的高为h(h >= 1)
    • 第i层的节点数量： 2^i-1^
    • 叶子节点的数量： 2^h-1^
    • 总结点
      • n = 2^h^ - 1 = 2^0^ + 2^1^ + 2^2^ + ... + 2 ^h-1^
      • h = log~2~(n+1)
    • 在同样高度的二叉树中，满二叉树的叶子节点数量最多、总节点数量最多
    • 满二叉树一定是真二叉树，真二叉树不一定是满二叉树

完全二叉树

叶子结点只会出现最后2层，且最后1层的叶子节点是靠左对齐

  • 度为 1 的节点只有左子树

  • 度为 1 的节点要么是 1 个，要么是 0 个

  • 同样节点数量的二叉树，完全二叉树的高度最小

  • 假设完全二叉树的高度为h(h>=1),那么

    • 至少有==2 ^h-1^==个节点

    • 最多有==2 ^h^ - 1== 个节点

    • 总节点数量为 n

      ✓ 2h − 1 ≤ n < 2h ✓ h − 1 ≤ log2n < h ✓ h = floor( log2n ) + 1

      向下取整的运算称为Floor，用数学符号⌊⌋表示；如：⌊6.565⌋ = 6

      向上取整的运算称为Ceiling，用数学符号⌈⌉表示 如:⌈432.4⌉ = 433

      叶子节点为n0，度为1 为n1,度为2为n2

      n = n0+n1+n2 n0 = n2+1

      n = 2n0+n1-1

  • 完全二叉树的 n1 要么为 0，要么为 1

    • n1为1时，n = 2n0，n 必然是偶数
      • 叶子节点个数 n0 = n / 2，非叶子节点个数 n1 + n2 = n / 2
      • n1为0时，n = 2n0 – 1，n 必然是奇数
      • 叶子节点个数 n0 = (n + 1) / 2，非叶子节点个数 n1 + n2 = (n – 1) / 2
      • n0 = floor((n+1)/2)
      • n0 = (n+1)>>1

  •  叶子节点个数 n0 = floor( (n + 1) / 2 ) = ceiling( n / 2 )  非叶子节点个数 n1 + n2 = floor( n / 2 ) = ceiling( (n – 1) / 2 )  因此叶子节点个数为 384

二叉搜索树（Binary Search Tree）

  • 二叉搜索树是的一种，应用非常广泛英文简称为 BST  又被称为：二叉查找树  任意一个节点的值都 大于 其左子树所有节点的值  任意一个节点的值都 小于 其右子树所有节点的值  它的左右子树也是一棵二叉搜索树

  • 二叉搜索树存储的元素必须具备可比较性  比如 int 、double 等  如果是自定义类型，需要指比较方式  不允许为 null

    public static interface Visitor<E> {
        void visit(E element);
    }
    public void levelOrder(Visitor<E> vistor) {
        if (root == null) return;
        Queue<Node<E>> queue = new LinkedList<Node<E>>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            Node<E> node = queue.poll();
            vistor.visit(node.element);
            if (node.left != null) {
                queue.add(node.left);
            }
            if (node.right != null) {
                queue.add(node.right);
            }
        }
    }

高度

递归形式

public int height() {
        return height(root);
    }

    private int height(Node<E> node) {
        if (node == null)
            return 0;
        return Math.max(height(node.left), height(node.right)) + 1;
    }

非递归形式

public int height2() {
        if (root == null) return 0;
        int height = 0;
        int levelSize = 1;
        Queue<Node<E>> queue = new LinkedList<Node<E>>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            Node<E> node = queue.poll();
            levelSize--;

            if (node.left != null) {
                queue.add(node.left);
            }
            if (node.right != null) {
                queue.add(node.right);
            }
            // 判断当前的层是否走完
            if (levelSize == 0) {
                levelSize = queue.size();
                height++;
            }
        }
        return height;
    }

前驱节点（predecessor）

中序遍历：

条件

  1. node.left = null
  2. predecessor = node.left.right.right...
  3. node.left == null && node.parent != null找上层在父节点右边的父节点

代码

    private Node<E> predecessor(Node<E> node) {
        if(node== null) return null;

        //前驱节点在左子树中
        Node<E> p = node.left;
        if(node.left != null) {
            while (p.right!=null) {
                p = p.right;
            }
            return p;
        }
        //从父节点，祖父节点中寻找前驱节点
        while(node.parent != null&&node == node.parent.left) {
            node = node.left;
        }

        //node.parent==null
        //node==node.parent.right
        return node.parent;    
    }

后继节点（successor）

条件：

  1. node.right= null
  2. predecessor = node.right.left.left...
  3. node.left == null && node.parent != null找上层在父节点左边的父节点
private Node<E> successor(Node<E> node) {
        if(node== null) return null;

        //前驱节点在左子树中
        Node<E> p = node.right;
        if(node.right != null) {
            while (p.left!=null) {
                p = p.left;
            }
            return p;
        }
        //从父节点，祖父节点中寻找前驱节点
        while(node.parent != null&&node == node.parent.right) {
            node = node.right;
        }

        //node.parent==null
        //node==node.parent.left
        return node.parent;

    }

检测完全二叉树

    public Boolean isComplete() {
        if (root == null) return false;
        Queue<Node<E>> queue = new LinkedList<>();
        queue.offer(root);

        boolean leaf = false;
        while (!queue.isEmpty()) {
            Node<E> node = queue.poll();
            if (node.hasTwoChildren()) {
                queue.offer(node.left);
                queue.offer(node.right);
            } else if (!node.isLeaf()) {
                return false;
            } else {
                leaf = true;//检验成功！
                if(node.left!=null) {
                    queue.offer(node.left);
                }
            }
        }
        return true;
    }
